CFX.ResourcePerformance.SolderPastePrinting Namespace

[This is preliminary documentation and is subject to change.]

Specialized messages related to the performance of solder paste printers. This might include information related to stencil or squeegee cleaning rates, etc.
Classes
  ClassDescription
CleanSqueegeeRequest
Allows an external source to direct a request to a stencil printer to perform a squeegee clean operation
Copy
{
  "CleanTypeRequested": "Deep"
}
CleanSqueegeeResponse
Response to a request from an external source for a squeegee clean operation to be performed
Copy
{
  "Result": {
    "Result": "Success",
    "ResultCode": 0,
    "Message": "OK"
  }
}
