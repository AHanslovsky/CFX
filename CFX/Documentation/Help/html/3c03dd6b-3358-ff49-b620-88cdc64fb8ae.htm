MessageVersion
The version number of the enclosed CFX message.
RequestID
For envelopes involved in asynchronous request/response transactions between endpoints, a unique identifier (128-bit GUID) that uniquely identifies the request/response transaction. The endpoint that initiates the "Request" generates this identifier, and endpoints that respond should include this identifier in their "Response".
TimeStamp
The date/time when the event associated with this message ocurred.
